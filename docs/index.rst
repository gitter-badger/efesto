Welcome to Efesto
=================

Efesto is a RESTful API generator based on
`falcon <https://github.com/falconry/falcon>`_. It allows you to
build awesome applications quickly and without headaches because it handles
authentication, permissions and the API itself.

Efesto has been developed with security, performance and ease-of-use in mind and 
tailors very well as backend to a broad range of applications, from to-do lists
to ecommerce.
